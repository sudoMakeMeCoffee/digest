Digest API Documentation

The Digest API allows you to compile and run code in different programming languages using Docker containers.

üìå POST /api/v1/execute

Sends code to be compiled and run in a sandboxed Docker container.

Request body:

{ "language": "python", "version": "3.11", "files": [ { "name":"main.py", "content": "print('Hello from Digest!')" } ], "stdin": "" }

Response:

{ "stdout": "Hello from Digest!", "stderr": "", "exitCode": 0 }
